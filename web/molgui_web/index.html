Controls

Left Click: Select Atom

Drag: Box Selection

Right Drag: Rotate View

Wheel: Zoom

Modifiers:

  • Shift + Click/Drag: Add to selection
  • Ctrl + Click/Drag: Subtract from selection
  • Click Background: Clear selection

Shortcuts:

  • G: Toggle Gizmo
  • T: Translate Mode
  • R: Rotate Mode
  • S: Scale Mode
  • Esc: Clear Selection

Usage & links
